                        SysRev
			======

SysRev is a web application for selection of papers during systemic reviews and
meta-analyses.

SysRev is written in HTML and JavaSrcipt and runs in the browser.

SysRev only supports bibliographies in the Zotero Better BibLaTeX JSON format.

Bibliography files can be loaded via the file browser in the start page.
Alternatively online files may be provided via URL parameters. URL parameters
are as follows:

src = link to the webpage providing the json file
file = name of the file on the server
doi = a DOI resolver (example: doi.org)
